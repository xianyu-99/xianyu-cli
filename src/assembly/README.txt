YuCLI - A Java Agent CLI
========================

Version: ${project.version}
Build Date: ${maven.build.timestamp}

Requirements
------------
- Java 17 or higher
- JAVA_HOME environment variable set (optional but recommended)

Installation
------------
1. Extract this archive to your desired location.
2. Add the `bin` folder to your system PATH, or run scripts directly.

Windows
-------
Run: bin\yucli.bat

Unix / Linux / macOS
--------------------
Run: bin/yucli.sh

First run:
  chmod +x bin/yucli.sh
  ./bin/yucli.sh

Configuration
-------------
YuCLI stores its configuration and data in the user home directory.
